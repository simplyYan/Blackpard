Actions (Functions)

Simple function

Functions are blocks of code that you can call at any time, not necessarily when the program is executed. In them you can do anything, and even return information. Here's an example of a function in Blackpard:


action hello():

As already mentioned, you can return information in functions. Here's an example:

Making your program much faster (PVM)

Introduction PVM, short for PardVM, is an acceleration library that allows you to cache the results of commonly used function calls. This increases efficiency by avoiding repeated calculations and speeding up program execution. PVM has been implemented in the C language and can be accessed and used easily through Blackpard's range of built-in libraries. Features PVM offers a single main function, pvm, which can be used to memoize any Blackpard function. This function caches the results of function calls, allowing previously calculated results to be returned directly from the cache without recalculating. - Advantages of using PVM: - Improved performance: Avoids unnecessary recalculations, saving computational time. - Reduced resource consumption: Stores previous results, saving memory and processing. - Ease of use: Simple integration with existing Blackpard functions. Example of use Here is a practical example demonstrating how to use PVM in Blackpard to accelerate a function: 


//Load the built-in library
if ctypes.sizeof(ctypes.c_voidp) == 8:
    lib = extends('./libmemoize.so')  //Linux
else:
    lib = extends('./memoize.dll')   //Windows

//Define argument and return types for the pvm function
lib.pvm.argtypes = [ctypes.CFUNCTYPE(ctypes.c_int, ctypes.c_int), ctypes.c_int]
lib.pvm.restype = ctypes.c_int

//Blackpard function to be memoized
action example_function(x):
    echo("Calculating result for:", x)
    return x * x

//Convert the Blackpard function to a type understandable by ctypes
@ctypes.CFUNCTYPE(ctypes.c_int, ctypes.c_int)
action wrapped_example_function(x):
    return example_function(x)

//Call the pvm function to memoize the Blackpard function
memoized_result = lib.pvm(wrapped_example_function, 5)
echo("Result of memoized function call:", memoized_result)

//Call the pvm function again with the same argument
memoized_result_again = lib.pvm(wrapped_example_function, 5)
echo("Result from memoized cache:", memoized_result_again)

Explanation of the Example 1. Load the shared library: Use ctypes to load the shared library (libmemoize.so or memoize.dll) containing the pvm function. 2. Define argument and return types: Set argument and return types for the pvm function to be called. 3. Define the Blackpard function: Create a Blackpard function (example_function) to be memoized. 4. Convert the Blackpard function: Convert the Blackpard function to a type understandable by ctypes. 5. Call the pvm function: Utilize pvm to memoize the Blackpard function and calculate the result for a specific argument. 6. Call the pvm function again: Retrieve the result from the cache for the same argument, avoiding recomputation.

Indentation is part of syntax

Indentation is a way of tidying up the code, making some lines more to the right than others as we add spaces at the beginning. For most languages, indentation is not compulsory, but Blackpard is different. Indentation is an important feature in Blackpard because, in addition to promoting readability, it is essential for the code to work properly. In other words, if the indentation is not adequate, the program may behave unexpectedly or even fail to compile. For example, in the case of an if, what determines whether a code is within the conditional is the fact that it has been indented. Code 7 shows an example where the print command will only be executed if the value of the variable x is greater than 8.


    x = 10

    if x > 8:
       print("x is greater than 8")
    
    Code 7. Print command inside an if
    
    Now look at this new example in which the print command will be executed anyway because it is not indented and therefore not contained in the if.
    
    x = 10
    
    if x > 8:
    
    echo("x is greater than 8")
    
    Code 8: Print command outside an if
    
    There are two errors in Code 8. The first has to do with the way the code was written, because if the print command is only supposed to be executed if x is greater than 8, it should be indented.

Operators

Operators are symbols that represent mathematical operations. The main ones are: +, -, *, / and **, which represent addition, subtraction, multiplication, division and exponentiation respectively. The following code shows an example of multiplication between hours and seconds:


    .int hour=60
    .int seconds=60
    echo('1 hour contains: ',hour * seconds, 'seconds')
    // output: 1 hour contains: 3600 seconds

When the expression uses more than one operator, the language will take into account the rule of precedence, similar to what happens in conventional mathematics. Therefore, parentheses have the highest precedence and can be used in the expression to produce a desired result, causing a certain operation to be performed before the others. Exponentiation is next in the order, i.e. 3**1+1 equals 4 and not 5, just as 2*1**3 equals 2 and not 8. Multiplication and division have the same precedence, higher than addition and subtraction, which also have the same precedence. So 2*3-1 is 5, not 4, and 6+4/2 is 8, not 5. When two operators have the same precedence, they are evaluated from left to right. In other words, 5-3-1 is worth 1, not 3. This is because 5-3 is first calculated and then the value of 1 is subtracted. If in doubt, use the brackets to make sure the expressions come out as desired.

Another widely used operator is the modulus operator, which works on integers and returns the remainder of dividing the first by the second term. In Blackpard, it is represented by the percentage symbol %. Listing 7 shows on line 1 the command to assign the result of dividing 7 by 3 to the quotient variable. The modulo operation is then performed on line 4, assigning the result to the remainder variable.

While

With the while loop we can execute a set of statements as long as a condition is true.


    i = 1
    while i < 6:
      print(i)
      i += 1

The break Statement

With the break statement we can stop the loop even if the while condition is true:


    i = 1
    while i < 6:
      echo(i)
      if i == 3:
        break
      i += 1

The continue Statement

With the continue statement we can stop the current iteration, and continue with the next:


    i = 0
    while i < 6:
      i += 1
      if i == 3:
        continue
      print(i)

For loop

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string). This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages. With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc.


    fruits = ["apple", "banana", "cherry"]
    for x in fruits:
      echo(x)

String formatting

String format() The format() method allows you to format selected parts of a string. Sometimes there are parts of a text that you do not control, maybe they come from a database, or user input? To control such values, add placeholders (curly brackets {}) in the text, and run the values through the format() method: Example Get your own Blackpard Server Add a placeholder where you want to display the price: price = 49 txt = "The price is {} dollars" echo(txt.format(price)) You can add parameters inside the curly brackets to specify how to convert the value: Example Format the price to be displayed as a number with two decimals: txt = "The price is {:.2f} dollars" Check out all formatting types in our String format() Reference. Multiple Values If you want to use more values, just add more values to the format() method: echo(txt.format(price, itemno, count)) And add more placeholders: Example quantity = 3 itemno = 567 price = 49 myorder = "I want {} pieces of item number {} for {:.2f} dollars." echo(myorder.format(quantity, itemno, price)) Index Numbers You can use index numbers (a number inside the curly brackets {0}) to be sure the values are placed in the correct placeholders: Example quantity = 3 itemno = 567 price = 49 myorder = "I want {0} pieces of item number {1} for {2:.2f} dollars." echo(myorder.format(quantity, itemno, price)) Also, if you want to refer to the same value more than once, use the index number: Example age = 36 name = "John" txt = "His name is {1}. {1} is {0} years old." echo(txt.format(age, name)) Named Indexes You can also use named indexes by entering a name inside the curly brackets {carname}, but then you must use names when you pass the parameter values txt.format(carname = "Ford"): Example myorder = "I have a {carname}, it is a {model}." echo(myorder.format(carname = "Ford", model = "Mustang"))
